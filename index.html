— 04 · Get Started —

Installed in three steps

Node.js 18+ is required. Need an OpenAI API key (gpt-4o-mini costs about $0.004 per release).

STEP 01

Install globally

Via npm, yarn, or pnpm.

npm install -g kronos-agent
STEP 02

Configure once

API key, language, platforms, tone.

kronos config
STEP 03

Ship anything

Done. Watch the magic.

kronos ship --release v1.0.0
— 05 · FAQ —

Questions, answered

How much does Kronos cost? +
Kronos itself is free and open-source. You only pay for OpenAI API usage (gpt-4o-mini). A typical release costs around $0.004. A $5 budget lasts for roughly 1,250 releases.
Do I need conventional commits? +
Not required, but highly recommended. Kronos works with any commit messages but produces better output with conventional commits like feat:, fix:, and BREAKING CHANGE:.
Which languages are supported? +
Seven languages: English (US & UK), French, Chinese, Russian, German, and Spanish. Content is written natively in each language — not machine-translated. More languages are on the roadmap.
Can I use Kronos in my CI/CD pipeline? +
Absolutely. Set the OPENAI_API_KEY environment variable and run kronos ship --release $VERSION. GitHub Actions example is in the README.
Can I customize the output? +
Yes — pick your tone (professional, casual, excited, technical), choose which platforms to generate for, and set your default language. Custom templates are on the roadmap.
Is my code sent to OpenAI? +
Only your commit messages are sent — never your source code. Kronos parses git log metadata (commit messages, authors, timestamps) and nothing else.
